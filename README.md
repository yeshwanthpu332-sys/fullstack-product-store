# ShopEase

A full-stack e-commerce application built with React and Node.js.

## Features
- Product listing
- Category filter
- Search
- Sorting
- Product details page
- Cart
- Wishlist
- User authentication

## Tech Stack

### Frontend
- React
- Tailwind CSS
- React Router
- Axios
- Context API

### Backend
- Node.js
- Express
- SQLite
- JWT Authentication
- Bcrypt

## Getting Started

### Backend

```bash
cd "product store backend"
npm install
node seedProducts.js
node server.js
```

### Frontend

```bash
cd "product store"
npm install
npm run dev
```
